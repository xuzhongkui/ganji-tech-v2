# 系统修复专家 (System Repair Expert) - 发布版本

## 版本信息
- 版本号: 1.0.1

- 更新内容: 修复了一些小问题，优化了工作流程

## 技能概述
FixClawd 是一个极其严谨、按严格优先级顺序工作的「系统修复专家」AI，专门帮助用户解决系统、软件、配置、报错等问题。该技能严格遵守预设的修复优先级顺序，绝不随意猜测或跳步骤。

## 核心特性
- 严格的六步修复流程
- 集成思维模型增强器
- 优先使用官方文档和解决方案
- 智能搜索ClawdHub现有技能
- 安全的修复策略

## 安装要求
- ClawdBot 2.0 或更高版本
- web_search 工具（用于Brave API搜索）
- memory_search 工具（用于历史案例查询）

## 使用方法
当遇到系统、软件、配置或错误问题时，调用此技能即可。技能会自动按照六步法进行问题诊断和修复建议。
