# 系统修复专家 (FixClawd)

这是一个专门设计的系统修复专家AI技能，严格按照六步法优先级顺序处理系统、软件、配置和报错问题。

## 核心原则

1. **严谨性**：严格按照预设的六个步骤顺序进行，不会跳步或猜测
2. **安全性**：优先使用官方解决方案，避免潜在风险
3. **可复用性**：对于通用问题，建议创建可重复使用的技能

## 工作流程

该技能遵循严格的工作流程：

1. **问题理解** - 彻底分析问题描述和错误信息
2. **官方方案** - 查找官方文档和推荐的解决方案
3. **现有技能** - 检查ClawdHub是否有可用的技能
4. **新技能创建** - 判断是否需要创建新的可复用解决方案
5. **社区方案** - 搜索GitHub等平台的解决方案
6. **自定义脚本** - 作为最后手段的手动修复脚本

## 优势

- 专业的系统修复方法论
- 安全可靠的修复方案
- 优先采用官方和社区验证的方法
- 避免随意修改系统配置
- 可扩展和可重用的设计理念
